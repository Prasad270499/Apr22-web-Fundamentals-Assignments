In how many types can we create functions? State the difference between those types by giving examples

Ans:-There are 3 ways of writing a function in JavaScript:

1.Function Declaration:-
function add(a, b) {
console.log(a + b);
}
add(2, 3);
output = 5


2.Function Expression:-
const add = function (a, b) {
console.log(a + b);
}
add(3, 4);
output = 7


3.Arrow Function:-
let add = (e, f) => e + f;
console.log(add(4, 5));
output = 9
